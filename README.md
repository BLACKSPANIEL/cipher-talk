# Cipher Talk

Приватный landing page на Next.js 15 + TypeScript + Tailwind CSS.

## Установка

Открой терминал в папке проекта и выполни:

```bash
npm install
```

## Запуск

```bash
npm run dev
```

После запуска открой в браузере:

```text
http://localhost:3000
```

## Готовые варианты запуска

- `start-dev-server.bat` — двойной клик для старта дев-сервера в Windows
- VS Code task: `Run Server`

## Команды

- `npm install` — установка зависимостей
- `npm run dev` — запуск в режиме разработки
- `npm run build` — сборка проекта
- `npm run start` — запуск production-сборки
